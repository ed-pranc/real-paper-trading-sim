# Feature Implementation

Always follow this exact flow:  
1. Read project-onboarding and relevant skills  
2. Analyse affected modules  
3. Share a short plan (2-4 bullets)  
4. Implement while keeping existing style  
5. Add or update tests  
6. Run full checks via guardian  
7. Give one-sentence summary + changed files only

Never change unrelated files.
