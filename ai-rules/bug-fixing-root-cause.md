# Bug Fixing & Root Cause

1. Reproduce the bug exactly  
2. Find the real cause (not just the symptom)  
3. Check logs, stack trace and related files  
4. Propose minimal fix  
5. Add regression test  
6. Explain why the bug happened  
7. Run guardian before closing
